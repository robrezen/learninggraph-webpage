Grafo de Aprendizagem para o Moodle

Projeto desenvolvido pela Escola Politécnica da Universidade de São Paulo.

Funcionalidades Principais

  • Geração de grafos de conhecimento específicos para cada aluno
  • Feedback personalizado baseado no desempenho
  • Integração completa com o ambiente Moodle
